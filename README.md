# MySkillSwapNow - Legal Onboarding Starter

This version adds a first-run onboarding / install acknowledgement screen so users must
review and accept core platform rules before entering the app.

Included:
- Terms summary shown on first open
- User responsibility acknowledgement
- Platform-not-a-party disclaimer
- Liability disclaimer
- Skill-credit escrow explanation
- Dispute / ratings explanation
- Acceptance checkbox + continue button
- Acceptance saved locally in browser storage

## Run locally
npm install
npm start

## Deploy
Upload to GitHub and import into Vercel.
